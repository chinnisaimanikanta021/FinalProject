# CareerPath 🚀

CareerPath is a modern, interactive web application designed to help users explore career opportunities, education levels, and specific job roles with a curated AI-powered assistant.

## ✨ Features

- **Education-Based Exploration**: Browse career paths starting from your current education level (e.g., Undergraduate, Postgraduate).
- **In-Depth Job Details**: View comprehensive information about various roles, including required skills, typical responsibilities, and growth paths.
- **AI Career Assistant**: A custom-built 'Ask AI' bot to answer your career-related queries in real-time.
- **Modern UI/UX**: Featuring a sleek glassmorphism design with a responsive and intuitive interface.
- **Authentication**: Secure login and signup functionality to personalize your experience.

## 🛠️ Tech Stack

- **Frontend**: React.js
- **Build Tool**: Vite
- **Styling**: Vanilla CSS (Modern CSS features)
- **Icons**: Lucide React
- **Authentication**: Context-based Auth API

## 🚀 Getting Started

### Prerequisites

- Node.js (v18 or higher recommended)
- npm or yarn

### Installation

1. **Clone the repository**:
   ```bash
   git clone https://github.com/chinnisaimanikanta021/CareerPath.git
   ```

2. **Navigate to the project directory**:
   ```bash
   cd CareerPath
   ```

3. **Install dependencies**:
   ```bash
   npm install
   ```

4. **Run the development server**:
   ```bash
   npm run dev
   ```

5. **Open your browser**:
   Navigate to `http://localhost:5173` to see the application in action.

## 📁 Project Structure

- `src/components/`: Reusable UI components (CareerBot, CardGrid, JobDetails, etc.)
- `src/context/`: Authentication and Global state management.
- `src/data/`: Curated career data and resources.
- `src/App.jsx`: Main application logic and routing.
- `index.html`: Entry point.
